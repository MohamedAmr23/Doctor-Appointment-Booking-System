# Prescripto - Doctor Appointment Booking System

Live Demo: https://doctor-appointment-booking-system-rust.vercel.app/

## Overview

Prescripto is a modern and user-friendly doctor appointment booking platform that allows patients to easily browse trusted doctors, filter by specialty, check availability, and book appointments online.

This project is an excellent addition to a frontend developer's portfolio, demonstrating real-world application development with clean UI, smooth booking flow, and responsive design.

## Key Features

- Browse and search doctors by specialty (e.g., General Physician, Gynecologist, Dermatologist, etc.)
- View detailed doctor profiles with qualifications, experience, and fees
- Interactive appointment booking with date and time slot selection
- User authentication (login/signup)
- My Appointments dashboard to view and manage booked appointments
- Fully responsive design (mobile & desktop friendly)
- Smooth navigation and intuitive user experience

## Tech Stack

- Frontend: React.js (with Vite)
- Styling: Tailwind CSS
- State Management: React Context API
- Routing: React Router
- Backend Integration: REST API (built with Rust)
- Deployment: Vercel
